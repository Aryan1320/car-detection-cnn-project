# Car Detection using CNN and Sliding Window

## Overview
This project implements car detection using a Convolutional Neural Network (CNN) trained from scratch and a sliding window technique.

## Features
- CNN model (no pre-trained weights)
- Sliding window detection
- Multi-class classification
- Bounding box detection

## Dataset
Dataset is provided as dataset.zip and extracted during runtime.

## How to Run
1. Upload dataset.zip
2. Run all cells
3. Upload test image

## Output
- Detects car using sliding window
- Draws bounding box around detected object
